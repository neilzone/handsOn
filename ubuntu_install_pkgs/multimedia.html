Multimedia Pakete¶

  • vlc : Der VLC Media Player

  • mpv : Der mpv ist ein Medienplayer.

    Der mpv ist eine Weiterentwicklung des MPlayer, der mit OpenGL und unter Nutzung der GPU eine verbesserte Videoqualität bietet als der MPlayer.

  • handbrake : HandBrake ist ein Video-Transcoder

  • openshot : OpenShot ist ein Viedeo-Editor

  • qmmp : Qmmp ist ein Qt basierter Multimedia Player

  • mixxx: Mixxx ist eine Disk Jokey Mix Software

  • audacious: Audacious ist ein (winamp like) Audioplayer

  • youtube-dl Kommandozeile für youtube:

    • youtube-dl

    • youtube-dl –help

    you

$ sudo -H ./scripts/ubuntu_install_pkgs.sh multimedia
