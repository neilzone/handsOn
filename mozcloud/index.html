Setup Mozilla Dienste¶

Hinweis

Seit der Apache auf WSGI mit Python 3 umgestellt wurde, können die Mozilla Dienste nicht mehr betrieben werden, da diese noch nicht Python 3 compliant sind. WIP: https://github.com/return42/moz-cloud

Die Mozilla-Cloud Dienste sind eine skallierbare Infrastruktur interoperabler Dienste. Einen Überblick liefert der Abschnitt Mozilla Cloud Services. Der Abschnitt Allgemeines zum Setup beschreibt die Umgebung in dem die Dienste der Mozilla-Cloud betrieben werden und der Abschnitt Sync-1.5 Server mit public accounts beschreibt, wie ein einfacher Firefox Sync-1.5 Server eingerichtet werden kann.
