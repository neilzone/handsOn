Docker¶

Docker vereinfacht die Bereitstellung von Anwendungen in Containern. Es basiert auf LXC besitzt jedoch inzwischen eine eigene API (libcontainer).

  • Was sind Container?

  • Linux Container (LXC & LXD)

Vergleicht man Docker mit LXC, so kann man vereinfacht sagen, dass man mit Docker eher Anwendungen virtualisiert, während man mit LXC eher Betriebsysteme virtualisiert, die allerdings alle den gleichen Kernel verwenden. Beides sind leichtgewichtige Virtualisierer, die keine Hardware emulieren (VM), sondern virtuelle Umgebungen (VE) über ein entsprechendes Rechte-Management im Kernel gegeneinander isoliert.
