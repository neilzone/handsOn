================================================================================
                                Bildbearbeitung
================================================================================

Es werden verschiedene Tools zur Bildbearbeitung installiert:

:deb:`gimp`
  High-End 2D *pixel-based* Bildbearbeitung

  - http://www.gimp.org/
  - gimp-data-extras zusätzliche Pinsel, Paletten und Gradienten
  - gimp-plugin-registry Depot mit optionalen Erweiterungen für GIMP

:deb:`cheese`
  Aufnahmen mit der *Webcam* https://wiki.gnome.org/Apps/Cheese

:deb:`pinta`
  Einfaches 2D *pixel-based* Zeichenprogramm http://pinta-project.com/

:deb:`inkscape`
  Einfaches Tool zur 2D *vector-based* Grafikbearbeitung
  https://inkscape.org/de/

:deb:`darktable`
  Software zur Aufbereitung und Verwaltung von Digitalfotos
  http://www.darktable.org/

:deb:`rawtherapee`
  RawTherapee ist ein RAW-Konverter zur Umwandlung und Bearbeitung von
  fotografischen Rohdaten von Digitalkameras in gängige Bildformate.
  http://rawtherapee.com/blog/screenshots

:deb:`gnome-photo`
  Foto Verwaltung aus dem GNOME Projekt https://wiki.gnome.org/Apps/Photos


Installation
============

.. code-block:: bash

   $ sudo -H ./scripts/ubuntu_install_pkgs.sh imgTools
